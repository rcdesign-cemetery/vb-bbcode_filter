Description:
------------

vBulletin addon. Help to cleanup disabled bbcodes. Especially good, when user copies text
from Rich Text editor (MS Office, Open Office) to forum WISYWIG.

1. Cleanup disabled bbtags from input text.

2. Cleanup disabled tags from parced output, if user permissions were changed later.

3. Allow override permissions for some tags on group basis: FONT, SIZE, COLOR

4. Have global setting, where to apply custom permissions
